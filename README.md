复制 `Organization ID`，粘贴到 `Claude 用户ID` 输入框，点击确认账号信息。

![Claude 充值页面粘贴 Organization ID 并确认账号](https://pic1.imgdb.cn/item/6a1b1f25172c6dd74dd73108.png)

页面显示账号和套餐无误后再点击 `确认充值`。完成后刷新 Claude 页面，检查 Pro 状态。

## 自己账号充值异常排查

### 当前账号不是常用 Claude 账号

先退出当前账号，重新登录你长期使用的 Claude 账号，再进入 Settings / Account 复制 `Organization ID`。

### Organization ID 看起来不完整

不要手动补字符。回到 Claude Account 页面重新复制，确认没有漏掉字符或多复制空格。

### Claude 用户ID 无法确认

检查输入框里是否填了邮箱、昵称或其他订单备注。`Claude 用户ID` 通常应填写 `Organization ID`。

### Pro 状态没有立即显示

刷新 Claude 页面或重新登录。确认页账号正确时，一般等待同步即可。

自己的账号能保留历史对话、项目资料和使用习惯。处理文档、代码和私人资料时，也比共享账号更适合长期使用。

### Claude 用户ID 和邮箱有什么区别？

邮箱用于确认登录账号，`Claude 用户ID` 通常填写 `Organization ID`。不要把邮箱直接填到用户 ID 输入框。

检查邮箱、`Organization ID`、套餐和订单金额。尤其是登录过多个账号时，要确认当前账号正确。

### Claude Pro 适合哪些任务？

适合长文档总结、文章改写、代码阅读、报告整理、资料分析和复杂问题拆解。
